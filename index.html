Bucles en lenguaje C


Las estructuras repetitivas se utilizan cuando se quiere que un conjunto de instrucciones se ejecuten un cierto número finito de veces, por ejemplo, escribir algo en pantalla cierta cantidad de veces.


Entrada 
                    #include {{ stdio }}
                    #include {{ stdlib }}
                    int main (){
                        printf("Hola {{ nombre }}\n");
                        printf("Hola {{ nombre }}\n");
                        printf("Hola {{ nombre }}\n");
                        return 0;
                    }
Salida 
                    Hola Juan
                    Hola Juan
                    Hola Juan

Ciclo for


La instrucción for explota la capacidad del ordenador para repetir procesos y para contar. Muchas veces esta instrucción se puede reemplazar por otras instrucciones o estructuras de tipo repetición.


Entrada 
                    #include {{ stdio }}
                    #include {{ stdlib }}
                    int main(){
                        int i; 
                        for(i = 0; i < 2 ; i++){
                            printf("Hola {{ nombre }}\n")
                            }
                            return 0;
                        }
Salida 
                    Hola Juan
                    Hola Juan
                    Hola Juan

Ciclo while


La instrucción while es una de las alternativas para la repetición de procesos en programación.
Como veremos, guarda cierta similitud con la instrucción for, hasta el punto de que en algunas ocasiones podrá optarse por el uso de una u otra de forma indistinta.


Entrada 
                    #include {{ stdio }} 
                    #include {{ stdlib }}
                    int main() {
                            int i=0; 
                            while (i < 2) {
                                printf("Hola {{ nombre }}\n");
                                i=i+1;
                            }
                            return 0;
                        }
Salida 
                        Hola Juan
                    Hola Juan
                    Hola Juan

Ciclo do-while


La instrucción do-while es guarda cierta similitud con la instrucción while con la diferencia que realiza la comparación posterior a ejecutar las instrucciones asignadas.


Entrada 
                    #include {{ stdio }} 
                    #include {{ stdlib }}
                        int main() {
                            int i=0; 
                            do{
                                printf("Hola {{ nombre }}\n");
                                i=i+1;
                            }while (i < 2);
                            return 0;
                        }
Salida 
                    Hola Juan
                    Hola Juan
                    Hola Juan
